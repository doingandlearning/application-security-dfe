# Module 7 — Exercises

This module has two core, discussion-friendly labs plus one optional lab.

- **Lab 7.1 — Warn vs block policy** (pipeline gate policy for sample findings)
- **Lab 7.2 — Minimum runbook: suspected credential leak** (developer-centred incident response)
- **Optional Lab 7.3 — Container security checklist** (build-time + runtime issues from a sample Dockerfile)

Each lab is designed to produce something teams can reuse immediately (a policy snippet, a runbook, a checklist).
